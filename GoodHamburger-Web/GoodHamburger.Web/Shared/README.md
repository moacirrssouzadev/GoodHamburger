# Shared Components, Services & Models

Componentes, serviços e modelos reutilizáveis por toda a aplicação.

## Estrutura

- **Components/**: Componentes Razor reutilizáveis (MainLayout, NavMenu, etc)
- **Services/**: Serviços compartilhados (HTTP, API, etc)
- **Models/**: DTOs e entidades compartilhadas
- **Helpers/**: Funções utilitárias
- **Extensions/**: Métodos de extensão

## Convenção de Nomenclatura

- Componentes: `PascalCase.razor`
- Serviços: `*Service.cs`
- Models: `PascalCase.cs`
